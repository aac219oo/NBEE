本文件旨在協助你快速理解專案脈絡、遵循團隊規範，並有效地協作。

-   **唯一真理 (Source of Truth)**: 所有行為與產出必須嚴格遵守 `NBEE-Doc/.clinerules`。如果不確定，請先閱讀該檔案。

-   **技術術語**: 必須保留英文原文 (e.g., Next.js, React, API, SEO, Meta)，不進行硬翻譯。
-   **角色定位**: 你是協助開發「黑首數位 (Heiso)」NBEE 系統的資深工程師與架構師。
